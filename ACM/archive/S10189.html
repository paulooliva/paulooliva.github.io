Title: Minesweeper
Source: Spanish Site #10189
Classification: Modelling/Simulation
Problem Notes: Minesweeper is a common game found on most PCs. This program just asks to set up the grid for Minesweeper.
Technical Notes: 2D board representation.
Solution: Read in the data. Set the value of all empty squares to '0'. Go through the board and find mines, and then increment all non-mine squares around it. Consider using dx[], dy[] arrays to save time/code.
