Title: Secret Research
Source: Spanish Site #00621
Classification: Direct - Simple parsing
Problem Notes: This problem is very trivial, however it introduces a simple parsing of strings. Although it would be simple to perform case analysis, a more general approach to parsing is more useful to try.
Technical Notes: Very simple top down recursive parsing
Solution: Case analysis: There are only 4 cases to consider, and the problem suggests that in the case of a tie, the first condition that matches is to be selected. Another alternative is to write a function which parses the string piece by piece. Given a left point and a right point, it returns the type of string between the indices. Used recursively, this solves many other parsing problems as well as this one.
