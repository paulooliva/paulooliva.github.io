Title: Light, more Light
Source: Spanish Site #10110
Arithmetic
Problem Notes: Cute little mathematical problem.
Technical Notes: Basic input/output.
Solution: The number of times the nth lightbulb will be toggled is equal to the number of factors n has (Euler phi function). However, instead of calculating the phi function, it is enough to determine whether or not there will be an odd or even number of factors. The only numbers with an odd number of factors though are squares. So just check if n is a square.
