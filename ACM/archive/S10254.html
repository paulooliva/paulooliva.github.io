Title: The Priest Mathematician
Source: Spanish Site #10254
Classification: Recursion
Problem Notes: Hanoi Towers with a twist - 4 poles instead of 3. Note that the problem does not ask for an optimal solution, but just one following the rules they specify
Technical Notes: Requires Big Numbers
Solution: A simple approach is to figure out the recurrence. Let the sequence A1,A2, ..., An represent the minimum number of moves required to move n disks with 4 poles following the rules they want. Let the sequence B1, B2, ..., Bn represent the minimum number of moves required to move n disks with 3 poles (An = 2^n -1). To find A[n], we take the minimum over all i < n of 2*A[i]+B[n-i]. These number get large fairly quickly, so BigNumbers are required.
