Title: Counterfeit Dollar
Source: Spanish Site #00608
Classification: Modelling/Simulation
Problem Notes: This is a very simple problem if approached in the right way. Case analysis is one way, however a brute force method is a simpler choice, which eliminates many boundary cases.
Technical Notes: Simple string manipulation
Solution: Solution 1: Case analysis:
For each weighing go through and set whether or not a coin is normal, possibily light, or possibly heavy.

Solution 2: Brute force:
There are only 12 coins, and one of them is either too heavy or too light. Try all possibilities, and check that the measurements correspond to the current guess. Stop when one is found.
