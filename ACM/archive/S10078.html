Title: The Art Gallery
Source: Spanish Site #10078
Classification: Geometry - convex polygons
Problem Notes: The definition of the art gallery matches that of a convex polygon. A polygon is convex if for any two points selected within the polygon, the line connecting the two points is completely within the polygon. Another way of looking at a convex polygon is that the interior angles are always less than 180°
Technical Notes: This problem involves Euclidean geometry, and requires some way of representing points. Usually this is done with a structure called a Point consisting of two double values x and y. This structure is used throughout the code archive.
Solution: Method 1: Convex Hull
In the code archive, there is code that finds the convex hull of any polygon. A convex hull can be though of the shape formed by a rubber band when placed around the polygon. In order to check if the art gallery has no critical point, we can basically compute the convex hull, and check to see if it matches the polygon we had originally.
Method 2: Interior angles
Since the interior angles of a convex polygon are all less than 180°, we can go around in the order as provided by input, and compute interior angles using trigonometry.
