Title: Unidirectional TSP
Source: Spanish Site #00116
Classification: Dynamic Programming
Problem Notes: This program is a good introduction to the concept of dynamic programming, as well as path recovery.
Technical Notes: Some 2D array manipulation, and fairly straightforward input output
Solution: Create two 2-D matrices. Process the table column by column from the left. Keep track of the minimum possible cost to get to the square, as well as the square or direction that was taken to get to that square. Scan through the last column to find the minimum. After finding a solution, backtrack recursively to find the path taken to get the minimum
