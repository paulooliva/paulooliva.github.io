Title: I Love Big Numbers
Source: Spanish Site #10220
Classification: Arithmetic - BigNumber multiplication
Problem Notes: This problem is quite easy to understand, and deals with elementary concepts, however it is easy to try approaches that are too complicated, or spend time looking for patterns, when instead the direct method is fairly straightforward and easy to code.
Technical Notes: This is a good introductory problem to the use of big number arrays, and the use of a lookup table to increase efficiency. It is also introduces some of the concepts used by the BigNumber routines in the code archive. Another idea would be to use the Java BigInteger routines.
Solution: One approach is to simulate the multiplication process using a large integer array, and build a table of solutions from 1 to 1000
