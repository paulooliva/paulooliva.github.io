Studying Algebraic Structures using Prover9 and Mace4

R. Arthan and P. Oliva

In this page the reader can find the supporting material for the book chapter "Studying Algebraic Structures using Prover9 and Mace4".

Obtaining Prover9 and Mace4

In order to run the scripts below the reader should install the command line version of Prover9 and Mace4. These can be downloaded from here. These require a Unix-like operating system such as Linux or Mac OSX.

Obtaining the Supporting Material

The scripts used in this book chapter can be downloaded via the following links:

  • Section 1 scripts: Semilattices and hoops
  • Section 2 scripts: Advanced identities in hoops

After downloading and unzipping these, the user should run make on one of the example folders. Run the following command to get a description of what the scripts do:

$ make

Run the following command to run all the scripts in that example folder:

$ make all

Please note that the section 1 scripts will run in just a few seconds, but the proof searches in the section 2 script can be expected to take about two hours. You need to run the scripts as described above to create the output files referred to in the book chapter.

The make files by default assume prover9 and mace4 are in /usr/local/bin, you can overwrite this by setting the environment variable LADRBINDIR. For instance, if you have prover9 and mace4 on $HOME/bin/ you need to first run:

$ export LADRBINDIR=$HOME/bin/
