https://www.tv-films.co.uk
Back in 2013, I got the idea of combining data about upcoming movies on TV (initially Freeview), into a single web app, where users could easily filter and search for the best upcoming movies. The idea was to pull data about upcoming shows from tvguide.co.uk, and combine it with movie reviews from rottentomatoes.com. Later I also included movie trailer from YouTube. The first version of website was done in Haskell, but soon I found that it was easier to develop this in Python and Django. The current website allows users to keep a list of movies they have already seen (able filter these out from the list), or receive email alerts about movies they wish to see, so that when the movie is next on TV they receive an alert.
